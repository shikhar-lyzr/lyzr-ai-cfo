# Report Format Guidelines

## Summary Format (3-5 bullets)
Use for quick status updates and chat responses.
- Lead with overall position: on/over/under budget
- List top 2-3 variances by impact
- End with recommended next step

## Detailed Format (Full Narrative)
Use for monthly variance reports.
- Opening: Overall performance vs plan
- Section per category: actual vs budget, variance %, driver
- Closing: Outlook and recommended actions
- Length: 300-500 words

## Board Format (Executive Summary)
Use for board decks and executive briefings.
- One-line status: Green/Yellow/Red
- 3 key metrics: total spend vs budget, largest variance, trend direction
- 2-3 bullet action items
- Length: Under 150 words

## Email Format
Use for follow-up communications.
- Subject: "Action Required — [headline]"
- Opening: What was found
- Body: Supporting data (2-3 data points)
- Ask: Specific questions (numbered)
- Deadline: When response is needed
